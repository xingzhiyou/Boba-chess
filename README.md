# 泡姆泡姆啵叭棋复刻

本项目为泡姆泡姆啵叭棋的非官方复刻，仅供学习与交流使用。

## 玩法简介

- 9x9 棋盘，支持两名玩家对战。
- 规则与原作类似，支持领地扩张与消除。
- 详细规则见代码注释。

## 运行方法

1. 安装依赖（需要 Python 3 和 numpy）：
   ```sh
   pip install numpy
   ```
2. 运行游戏：
   ```sh
   python game.py
   ```
